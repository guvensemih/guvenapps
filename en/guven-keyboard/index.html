Güven Braille Keyboard

Güven Braille Keyboard is an Android keyboard app designed to enable blind and visually impaired users to type in Braille on a touchscreen.

About the app

The app is being developed to support Braille input, text editing and an accessible experience compatible with screen readers.

Güven Braille Keyboard is currently in development and testing. Accessibility, straightforward use and reliable text input are the main priorities.
